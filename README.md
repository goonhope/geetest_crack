## 通过率

- 滑动: 通过算法生成轨迹
  - 二代: 重试一次通过率99%
  - 三代: 不重试, 通过率99%
- 汉字点选: 99%

## 耗时
- 滑动: 4s以内
- 汉字点选: 10s以内(CPU上YOLO3比较耗时)

## 开发环境

- python3.6
- tensorflow
- keras
- darknet(YOLO3)
- labelImg(YOLO3数据标签工具)
- opencv(定位滑动缺口距离)
- pyppeteer

## 算法

- YOLO3: 定位汉字位置
- CRNN: 校验文字识别
- CNN: 定位后的文字识别

## 数据集

### 汉字点选

- 4000+汉字
- 校验文字样本约50K
- YOLO3样本3.6K
- 定位后的文字样本140K
